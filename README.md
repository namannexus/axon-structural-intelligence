# 🏗️ AXON: Autonomous Structural Intelligence

AXON is an AI-driven system that transforms 2D floorplans into queryable 3D geometric graphs, providing real-time material optimization and engineering reports.

## 🚀 Key Features
- **CV Pipeline:** OpenCV-based geometry extraction and wall classification.
- **3D Visualization:** Interactive reconstruction using React Three Fiber.
- **Material Engine:** Optimization logic for 30+ construction materials.
- **Auto-Reporting:** Instant PDF generation with structural risk analysis.

## 🛠️ Technical Stack
- **Frontend:** React, Tailwind CSS, Three.js (R3F)
- **Backend:** FastAPI (Python), OpenCV, FPDF2
- **Intelligence:** LLM Explainability Layer for structural reasoning.

## 📦 Quick Start
1. **Backend:** `cd backend && pip install -r requirements.txt && python main.py`
2. **Frontend:** `cd frontend && npm install && npm run dev`
